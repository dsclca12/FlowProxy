# Security Policy

## Supported Versions

Only the latest release line is supported with security fixes.

| Version | Supported |
| --- | --- |
| v0.2.x (latest patch) | Yes |
| older versions | No |

## Reporting a Vulnerability

Please do not open public issues for undisclosed vulnerabilities.

Preferred channel:
1. Open a private GitHub Security Advisory for this repository.

Fallback:
1. Open a minimal public issue titled `Security contact request` without technical details.
2. A maintainer will provide a private follow-up channel.

Include:
- Affected version/commit
- Reproduction steps or PoC
- Impact assessment
- Suggested fix (if available)
