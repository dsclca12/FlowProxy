# Code of Conduct

## Our Standards

Participants in this project are expected to:

- Be respectful and constructive.
- Assume good intent and discuss technical disagreements with evidence.
- Focus feedback on code and behavior, not individuals.
- Avoid harassment, discrimination, or personal attacks.

## Unacceptable Behavior

- Harassment, hate speech, or abusive language.
- Publishing private information without consent.
- Trolling, intimidation, or repeated bad-faith interactions.

## Enforcement

Maintainers may remove comments, close discussions, or restrict participation
for behavior that violates this Code of Conduct.

## Scope

This policy applies to repository discussions, issues, pull requests, and
other project-managed communication channels.

## Reporting

For conduct concerns, open an issue titled `Conduct concern` with only the
minimum details needed to initiate private follow-up.
